**Java规范的注释方法：** 

- 单行注释
- 多行注释
- 文档注释（Java特有）



## 单行注释

```java
// 这是一个单行注释
```



## 多行注释

```Java
/*
这是一个多行注释
*/
```

多行注释不可嵌套使用。

## 文档注释（Java特有）

```java
/**
文档注释
@author Regina23
@version v1.0
*/
```

注释内容可以被JDK提供的工具javadoc所解析，生成一套以网页文件形式体现的程序说明文档。
